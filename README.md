✈️ Umanda
개인화 맞춤 여행지 추천 및 커뮤니티를 통한 동행 서비스



🍪 프로젝트 소개
여행자의 취향을 기반으로 유럽의 관광지 경로를 추천해주고 커뮤니티를 통해 동행 서비스를 활성화

기능 소개
동행 버튼을 통해 여행메이트를 찾을 수 있도록 구현
AI경로 추천을 통해 내 취향의 여행지를 추천받을 수 있도록 구현
만남 버튼을 통해 주변 여행객들과의 채팅 기능 구현
